Projet SAE 302 — Gestion des Sauveteurs Spéléologues

Application PyQt de gestion des équipes de secours du Spéléo-Secours Français

Auteurs

Pape Birame Fall

Sokhna Gueye

Perceval Behanzin

Mohamed Fall

Professeurs responsables

Benoit Tremblais

Objectif

Créer une application graphique permettant de gérer les sauveteurs spéléologues, leur planning, leurs missions et leur disponibilité, avec synchronisation pair-à-pair sur un réseau local.

Répartition des tâches

Pape Birame Fall — Gestion de la base de données & backend

  • Répartition des tâches et page html statique
  • Responsable du modèle de données (couche “Modèle” du MVC)
  • Écrire le script SQL : script_base_de_donne.sql
  • Créer les fonctions Python d'accès (CRUD)
  • Initialiser la base (sauveteurs.db) depuis Python
  • Gérer import/export JSON des sauveteurs
  • Fournir API Python : ajouter_sauveteur(), get_sauveteurs(), etc.

Sokhna Gueye — Interface graphique PyQt (Vue)

  • Créer fenêtres PyQt6 : authentification, planning, fiche récapitulative
  • Gérer interactions (boutons, menus, tableaux)
  • Intégrer couleurs du planning selon statuts

Perceval Behanzin — Logique applicative & synchronisation (Contrôleur)

  • Connecter la base aux interfaces via le module de Pape Birame Fall
  • Gérer mises à jour du planning et statuts en temps réel
  • Préparer communication réseau TCP (port 5002)
  • Gérer profils utilisateurs (lecture, gestionnaire, admin)

Mohamed Fall — Documentation & tests

  • Mettre en place Sphinx (docs/)
  • Rédiger .rst : installation, utilisation, description
  • Créer tests unitaires (tests/) pour modèle et contrôleur

Fonctionnalités principales

  • Gestion des sauveteurs et de leurs spécialités
  • Gestion du planning et des statuts colorés
  • Interface graphique PyQt simple et claire
  • Synchronisation automatique entre plusieurs postes via TCP (port 5002)
  • Authentification par profils : gestionnaire, lecture, administration

Technologies utilisées

  • Python 3.10+
  • PyQt5 pour l’interface graphique
  • SQLite (base locale)
  • Communication socket TCP
  • Documentation Sphinx
